Програма перевіряє чи користувач ввів у поля:
1. 2 слова
2. Довжину слів, більша за 2 (чи поле заповнили)
3. Номер телефону містить 10 цифр, чи 12 цифр і знак '+' на початку
4. Чи створюютть користувача, чий номер вводили до того (перевіряється чи існує такий користувач у базі даних)
